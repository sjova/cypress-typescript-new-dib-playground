## Tests Naming Examples

```ts
it('should error for an invalid password for existing user');

it('gets a list of users');
it('get users by email');
it('creates a new user account');
it('updates a user account');
it('deletes a user account');

it('displays new transaction errors');
it('shows pagination');
it('renders an empty notifications state');

it('toggles the navigation drawer');
it('filters articles by tag');
it('searches for a user by attribute');

it('submits a transaction payment');
it('login as user');

it('can follow second user');
it('error when invalid user email');
```
